Features

Everything you need to
control AI spend.

From real-time dashboards to prompt-level cost analysis. Built for engineering teams shipping AI products.

Real-Time Cost Streaming

Server-Sent Events deliver cost updates the instant they happen. Watch your spending as it accumulates with sub-second latency.

Feature-Level Breakdown

Tag every API call with a feature name. Know exactly which product feature is burning through your budget — chat, search, RAG, agents, etc.

Interactive Flamegraph

Hierarchical cost visualization. Drill from project to feature to model. Click to zoom, instantly spot the biggest cost centers.

Prompt Bloat Detection

Automatically identifies duplicate and overweight prompts. Find prompts costing you the most and optimize them for up to 40% savings.

Zero Code Changes

The SDK uses JavaScript Proxy to intercept API calls transparently. Wrap your client once — every call is tracked automatically, including streaming.

CSV & JSON Export

Export events and cost summaries with one click. Filter by date, feature, model, or provider. Feed data into your BI tools or spreadsheets.

Get Started in 60 Seconds

Ready to take control?

Clone the repo, spin up with Docker Compose, and start tracking costs immediately.

# Clone & start the stack
git clone https://github.com/your-org/ai-cost-profiler
cd ai-cost-profiler
docker compose up -d

# Install the SDK in your project
pnpm add @ai-cost-profiler/sdk

# Open the dashboard
open http://localhost:3000
